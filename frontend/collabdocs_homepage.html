features

everything your team needs to write better, together

built for speed, built for collaboration — not just another doc editor.

real-time editing
multiple people edit the same document simultaneously using Yjs CRDT — zero conflicts.
smart workspaces
organise documents into workspaces. invite your team with role-based access control.
roles & permissions
viewer · commenter · editor · admin — full RBAC so the right people see the right things.
version history
every change is snapshotted. restore any past version in one click, see who changed what.
inline comments
select any text and add a comment. mention teammates with @name. resolve when done.
rich block editor
headings, bullets, todos, code blocks, tables, images — type / for quick commands.
how it works

up and running in under 2 minutes

no setup, no installs — just open and collaborate.

1
create a workspace
sign up with Google or email. name your workspace, pick an icon, set it private or team.
2
invite your team
send invite links by email or share a workspace link. assign roles — editor, admin, or viewer.
3
write together, live
create documents, edit in real-time, comment, and never lose a version. everything syncs instantly.
